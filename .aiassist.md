# MAHE AI Assistant - Project Context

## Tech Stack
- Python 3.12
- Click + Rich + prompt-toolkit (CLI)
- LiteLLM (multi-model LLM integration)
- Custom ReAct Agent architecture

## Architecture
- `src/mahe/cli/` - User interaction layer (REPL, rendering)
- `src/mahe/core/` - Core engine (agent loop, orchestrator)
- `src/mahe/context/` - Context management (memory, project config)
- `src/mahe/tools/` - Tool system (file, shell, git, code)
- `src/mahe/llm/` - LLM integration (LiteLLM adapter)
- `src/mahe/utils/` - Utilities (permissions, sandbox, errors)

## Conventions
- All tools are async by default
- Follow ReAct pattern (Reasoning + Acting)
- Permission system: safe → confirm → blocked
- Message format follows OpenAI function-calling convention
- Use type hints everywhere
- Docstrings for all public classes and methods
